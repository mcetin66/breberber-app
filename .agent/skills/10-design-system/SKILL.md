# Design System - UI/UX Rehberi

> 8-point grid, typography scale, color system ve responsive breakpoints.

## Spacing (8-point Grid)

```
4px  = 0.25rem = space-1
8px  = 0.5rem  = space-2
16px = 1rem    = space-4
24px = 1.5rem  = space-6
32px = 2rem    = space-8
```

## Typography Scale

```
xs:   12px / 16px
sm:   14px / 20px
base: 16px / 24px
lg:   18px / 28px
xl:   20px / 28px
2xl:  24px / 32px
```

## Key Principles

- 8-point grid sistemi
- Mobile-first responsive
- Semantic color system
- Accessibility (WCAG 2.1 AA)
- Consistent spacing
